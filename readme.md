# CK Every Place

r/place but forever

## Features
 - Main 30x30 canvas with 36 colors
 - Create your own canvas(place), up to 1080x1080
 - Find and sub to other canvases
 - Private canvases
 - More!

## Technologies
  - PostgreSQL
  - Redis
  - Express.js backend-for-frontend
    - REST API
    - socket.io
  - Create-react-app frontend
